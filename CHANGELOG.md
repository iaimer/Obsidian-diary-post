# 更新日志

## [0.6.7] - 2026-05-15

### 新增功能
- **快速记录文本框自适应**：输入框随文字增多自动扩展高度，最大高度 200px
- **觉察和小确幸添加时间戳**：与随手记统一 `**HH:MM**` 时间戳格式
  - 觉察：`- **HH:MM** 内容 #标签`
  - 小确幸：`> **HH:MM** 内容 #标签`
  - 前端本地和服务端远程模式均已支持
- **觉察标签独立样式**：时间戳黄色粗体、标签黄色圆角 `bg-yellow-100 text-yellow-700`
- **小确幸标签独立样式**：引用块提取 `#` 标签并渲染为绿色圆角 `bg-green-50 text-green-600`

### Bug修复
- **觉察和小确幸保存时标签丢失**：修复 handleSubmit 未将选中标签传递给 DataService 的问题
  - DataService 接口及 Local/Remote 实现全部添加 tags 参数
  - fileSync 层添加 tags 格式化支持
  - 服务端 happiness/reflection 路由支持 tags 参数
  - DiaryView 和 DiaryDetail 渲染同步更新

### 文档更新
- **关于页**：荔枝喵说状态更新为"APP 内 AI 生成（待实现）"

## [0.6.6] - 2026-05-15

### 新增功能
- **图片上传与浏览器端压缩**：支持从手机/电脑上传照片到日记影像记录
  - Canvas API 浏览器端压缩，长边最大 2000px，JPEG 质量可调
  - 压缩目标 < 2MB，逐步降质量 + 降分辨率兜底
  - 上传后自动写入 Vault assets 目录并追加 WikiLink 引用
- **远程模式图片上传**：手机端通过 API Server 远程上传
  - base64 编码传输，服务端写入 assets + 追加 WikiLink
- **图片压缩设置**：可自定义压缩参数
  - 最大长边 (px)、最大文件大小 (MB)、JPEG 质量
  - 自定义文件名格式（占位符 `{date}` `{seq}`）
  - 设置持久化到 localStorage
- **今日日记图片大图查看**：点击影像记录缩略图打开 ImageModal 全屏弹窗
  - 与历史页面一致的交互体验

### 功能改进
- **日历格子优化**：记录数文字替换为日期下方蓝色小点
- **上传按钮**：右对齐，"添加照片"文字，无 emoji
- **图片命名**：`Image-YYYYMMDD-序号.jpg` 格式，自动扫描避免重名

### Bug修复
- **日历今天有图片时日期数字不显示**：修复 isToday + imageUrl 时缺少 text-white 样式
- **影像记录刷新问题**：上传后通过 triggerRefresh 机制确保 DiaryView 重新加载

## [0.6.5] - 2026-05-14

### 新增功能
- **觉察和小确幸润色**：与随手记统一润色功能
  - 添加润色按钮、标签选择器、AI标签显示
  - 支持三层标签体系（领域+能力+方法）
  - 润色预览、继续编辑、取消功能
- **自定义润色规则**：设置页面可自定义润色提示词
  - 默认润色规则：尊重事实零增补 + 适度修辞 + 轻微扩写
  - 可编辑 textarea 自定义规则内容
  - 重置为默认规则按钮
- **统一润色规则**：随手记、觉察、小确幸使用相同润色规则
  - 尊重事实零增补：不添加未提及的人物、事件、地点
  - 适度修辞：可适当用比喻、拟人，基于原文信息处理
  - 轻微扩写：可做1-2句扩写，仅延伸原文氛围/情绪

### 功能改进
- **ReflectionModal**：添加润色功能、标签选择器、标签显示
- **HappinessModal**：添加润色功能、标签选择器、标签显示

## [0.1.0] - 2026-05-09

### 首次发布

#### 核心功能
- **随手记输入**：快速记录日常事件
  - 三层标签选择器（领域+能力+方法）
  - 自动获取时间戳
  - 直接追加到Obsidian日记文件
- **觉察与小确幸**：快速输入弹窗
  - 觉察区块：追加到"💡 觉察与迭代"
  - 小确幸区块：追加到"✨ 每日小确幸"
- **习惯打卡**：5项习惯追踪
  - 饮水(mL)、运动(步)、阅读、学语言、补充剂
  - 目标提示：饮水≥1500mL、运动≥6000步为达标
- **日记查看**：分区块展示
  - 随手记、小确幸、觉察、荔枝喵说
  - 阅读模式渲染（无Markdown标记）
  - 标签显示在记录下方（中性灰颜色）

#### 技术特性
- **Obsidian Vault集成**：File System Access API
  - 路径格式：YYYY/序号.英文月份名/YYYY-MM-DD.md
  - 安全追加模式：先读再追加，不覆盖
- **离线缓存**：IndexedDB (Dexie.js)
- **自动刷新**：添加记录后自动更新显示
- **状态持久化**：localStorage保存连接状态和习惯数据

#### 技术栈
- React + TypeScript + Vite
- Tailwind CSS
- Zustand状态管理
- Dexie IndexedDB
