# 更新日志

所有重要的更改都将记录在此文件中。

## [0.2.0] - 2026-05-09

### 新增功能
- **设置页面**：支持完全自定义API配置
  - 服务名称自定义
  - Base URL自定义（支持任意API地址）
  - 模型名称自定义
  - API Key输入（支持显示/隐藏）
- **预设模板**：一键填充常用API配置
  - Claude API
  - OpenAI API
  - DeepSeek
  - Moonshot
  - 本地Ollama
- **AI润色引擎**：自动润色内容并添加标签
  - 润色规则：生动化扩写、事实零增补、拒绝代写总结、口语化、简短有节奏
  - 自动根据标签规范添加三层标签（领域+能力+方法）
  - 支持Claude API和OpenAI兼容API两种格式

### 功能改进
- **习惯追踪**：快捷按钮改为累加模式
  - 饮水：-250/+250/+500/目标/清零
  - 运动：-1000/+1000/+3000/目标/清零

## [0.1.0] - 2026-05-09

### 首次发布

#### 核心功能
- **随手记输入**：快速记录日常事件
  - 三层标签选择器（领域+能力+方法）
  - 自动获取时间戳
  - 直接追加到Obsidian日记文件
- **觉察与小确幸**：快速输入弹窗
  - 觉察区块：追加到"💡 觉察与迭代"
  - 小确幸区块：追加到"✨ 每日小确幸"
- **习惯打卡**：5项习惯追踪
  - 饮水(mL)、运动(步)、阅读、学语言、补充剂
  - 目标提示：饮水≥1500mL、运动≥6000步为达标
- **日记查看**：分区块展示
  - 随手记、小确幸、觉察、荔枝喵说
  - 阅读模式渲染（无Markdown标记）
  - 标签显示在记录下方（中性灰颜色）

#### 技术特性
- **Obsidian Vault集成**：File System Access API
  - 路径格式：YYYY/序号.英文月份名/YYYY-MM-DD.md
  - 安全追加模式：先读再追加，不覆盖
- **离线缓存**：IndexedDB (Dexie.js)
- **自动刷新**：添加记录后自动更新显示
- **状态持久化**：localStorage保存连接状态和习惯数据

#### 技术栈
- React + TypeScript + Vite
- Tailwind CSS
- Zustand状态管理
- Dexie IndexedDB
